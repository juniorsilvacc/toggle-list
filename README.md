# Como executar o projeto

## Pré-requitos: npm/yarn 

```bash
# clonar repositório  
git clone https://github.com/juniorsilvacc/toggle-list.git

# entrar na pasta do projeto mobile
cd reduxapp

# instalar dependências
yarn install

# executar projeto
yarn start
```
